# Cyvex Installation Guide

## Quick Start

### Step 1: Download Cyvex
1. Download or clone this repository to your computer
2. Extract the files to a folder (e.g., `cyvex-extension`)

### Step 2: Enable Developer Mode in Chrome
1. Open Google Chrome
2. Navigate to `chrome://extensions/`
3. Toggle "Developer mode" ON in the top-right corner

### Step 3: Load the Extension
1. Click "Load unpacked" button
2. Select the folder containing the Cyvex extension files
3. Click "Select Folder"

## Configuration

### Access Settings
1. Right-click the Cyvex icon in your toolbar
2. Select "Options" from the context menu
3. Or visit `chrome://extensions/` and click "Extension options"

### Available Settings
- **Auto-scan**: Enable/disable automatic link scanning
- **Security overlay**: Show/hide the security status overlay
- **Block dangerous links**: Show warnings before accessing dangerous links
- **Flag shortened URLs**: Mark shortened URLs as suspicious
- **Suspicious patterns**: Check for typosquatting and other patterns
- **Security tips**: Show/hide cybersecurity awareness tips

## Troubleshooting

### Extension Not Working
1. Check that Developer mode is enabled
2. Verify the extension is enabled in `chrome://extensions/`
3. Try refreshing the webpage
4. Check the browser console for errors (F12)

### No Link Indicators
1. Ensure auto-scan is enabled in settings
2. Try refreshing the page
3. Check if the page has any links to scan
4. Verify the extension has permission to access the site

### Database Not Updating
1. Check your internet connection
2. Try manually updating the database in settings
3. Clear the cache and try again
4. Check if your firewall is blocking the extension

### Performance Issues
1. Disable auto-scan for very large pages
2. Clear the extension cache
3. Restart Chrome
4. Check available system memory

## Uninstallation

### Remove Extension
1. Go to `chrome://extensions/`
2. Find Cyvex in the list
3. Click "Remove"
4. Confirm removal

### Clear Data (Optional)
1. Go to `chrome://settings/content/all`
2. Search for "Cyvex"
3. Click on the extension
4. Click "Clear data"
